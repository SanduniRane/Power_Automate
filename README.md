# 🚀 Invoice Automation using Power Automate Desktop

## 📌 Overview

This project demonstrates how to automate invoice creation using **Power Automate Desktop** with the **Contoso Invoicing Application**.

The goal of this automation is to reduce manual data entry, improve accuracy, and save time by streamlining repetitive invoicing tasks.

---

## ⚙️ Features

* Launches the Contoso Invoicing application automatically
* Navigates through the UI using recorded actions
* Populates invoice details:

  * Date
  * Account
  * Contact
  * Amount
* Selects invoice status (e.g., *Uninvoiced*)
* Completes the invoice process with minimal human interaction

---

## 🛠️ Technologies Used

* Microsoft Power Automate Desktop
* UI Automation (RPA)
* Windows-based application automation

---

## 📂 Workflow Summary

1. Run the Contoso Invoicing application
2. Navigate to the invoice section
3. Input required data into text fields
4. Select appropriate options from dropdowns
5. Submit and finalize the invoice

---

## 🎯 Purpose

This project is designed to:

* Demonstrate Robotic Process Automation (RPA) capabilities
* Showcase practical use of Power Automate Desktop
* Improve efficiency in business processes
